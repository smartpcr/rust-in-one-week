# basics

Demonstrates core Rust language fundamentals including:

- Integer overflow behavior: `checked_div`, `wrapping_mul`, `overflowing_sub`, `overflowing_add`
- Signed vs unsigned integer operations and their edge cases
- Floating-point special values (`INFINITY`, `MIN`, `MAX`)
- Character conversion with `from_digit`
- Vectors and the Padovan sequence
- Struct definition and iteration over references

## Run

```bash
cargo run -p basics
```
