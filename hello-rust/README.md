# hello-rust

Introductory Rust program demonstrating:

- External crate usage (`ferris-says` for ASCII art, `chrono` for date/time)
- Module system with `mod` declarations
- Struct implementation with methods (`Person` struct with `age()` calculation)
- GCD (Greatest Common Divisor) recursive implementation with unit tests
- Visibility modifiers (`pub(crate)`)

## Run

```bash
cargo run -p hello-rust
```

## Test

```bash
cargo test -p hello-rust
```
